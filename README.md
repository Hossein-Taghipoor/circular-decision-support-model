# Circular Decision Support Model

This project explores how to choose between repairing, replacing, or recycling a product based on its lifecycle characteristics.

The model uses a Kaggle dataset with carbon-related parameters across different stages of a product’s lifecycle, including energy use, emissions, transport, and usage conditions. It estimates total energy use and emissions to compare the different options.

# Key Insight

The results show that recycling is often preferred due to its lower production impact, especially when energy use or carbon intensity is high. However, repair becomes more favorable in lower-impact conditions. This highlights that the best strategy is not fixed, but depends on how and where the product is used.

# Takeaway

This project demonstrates how lifecycle data can be used to support circular decision-making, showing how trade-offs between cost, energy use, and environmental impact influence the choice between repair, replacement, and recycling.
